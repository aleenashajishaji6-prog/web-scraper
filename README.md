# Automated Web Scraper

A Python web scraping project built with BeautifulSoup, Selenium, SQLite, and a scheduler.

## Features
- Scrapes quotes and books from websites
- Handles both static and JavaScript-loaded pages
- Stores data in SQLite database and CSV files
- Runs automatically on a schedule

## Technologies
- Python, BeautifulSoup4, Selenium, SQLite, Schedule

## How to run
pip install requests beautifulsoup4 selenium webdriver-manager schedule
python scraper.py
